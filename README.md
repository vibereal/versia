# Versia Tuban

Versia Tuban adalah layanan jasa jahit custom pria premium yang berlokasi di Tuban. Kami mengkhususkan diri dalam pembuatan jas, kemeja, batik, dan celana dengan kualitas terbaik.

## Fitur Utama
- **Jahitan Presisi**: Pas di badan untuk kenyamanan maksimal.
- **Terpercaya**: Berpengalaman sejak 1990.
- **Cepat**: Layanan pengerjaan cepat dengan hasil memuaskan.
- **AI Preview Batik**: Coba preview batik menggunakan teknologi AI.
- **Konsultasi Mudah**: Konsultasi langsung via WhatsApp.

## Teknologi yang Digunakan
- Vite
- React
- TypeScript
- Tailwind CSS
- Shadcn UI
